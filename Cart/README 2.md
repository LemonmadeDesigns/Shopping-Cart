# How to Start Mongo on Mac

## Start MongoDB

```yaml
sudo service mongodb start
```

## Stop MongoDB

```yaml
sudo service mongodb stop
```

## Restart MongoDB

```yaml
sudo service mongodb restart
```

## To use MongoDB run the following command

```yaml
mongo
```

This will connect you to running MongoDB instance
